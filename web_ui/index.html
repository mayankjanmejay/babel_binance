Voting Configuration

Require a minimum number of algorithms to agree on the same direction (BUY/SELL) before executing a trade.

How many algorithms must agree (out of total enabled)

Example:

  • If 3 algorithms are enabled and required votes = 2
  • Trade executes when at least 2 algorithms signal BUY or SELL
  • Confidence is averaged from agreeing algorithms

Weighted Configuration

Each algorithm contributes to the final signal based on its weight. Weights must sum to 100%.

Example:

  • SMA: 40%, RSI: 35%, Grid: 25%
  • If SMA signals BUY (0.8 confidence), RSI signals SELL (0.6 confidence), Grid signals BUY (0.7 confidence)
  • BUY score: 0.4×0.8 + 0.25×0.7 = 0.495
  • SELL score: 0.35×0.6 = 0.21
  • No trade (BUY score below 0.5 threshold)

Unanimous Mode

All active algorithms must agree on the same direction (BUY or SELL) for a trade to execute. This is the most conservative mode with the fewest trades but highest confidence.

Characteristics:

  • Highest precision, lowest frequency
  • Best for risk-averse strategies
  • Averages confidence from all algorithms
  • Any disagreement = no trade
